# BLOGPOST CMS API

CMS api for smiple blog posts. Api can be connected to GUI which would allow to create content management system.

# Tech stack

- yarn2
- express
- awilix
- heroku
- postgress
- dotenv
- pg

# Install

This project was made using yarn2 and for that reason it is recommended to install [yarn].

However, feel free to use [npm] instead.

For this project you will also need a postgress database. Follow the [link][postgress] to see how to setup and connect to postgress.

Setup:

1. clone the repo: https://github.com/RihardsJ/blogpost-cms-api.git
2. `yarn install` or `npm install`
3. create `.env` file and add your postgress DB credentials
4. ...

# Further development

1. Create gui for the api
2. tables and relations can be added and customized to your blog post needs
